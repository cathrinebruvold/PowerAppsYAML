# powerapps-yaml-components

YAML Components for building applications efficiently and reusing beautiful app components.

## About

This repository contains YAML source files for **Power Apps Canvas App** components built on Microsoft Power Platform. These reusable components can be imported into your own Canvas Apps to accelerate development.

## Usage

1. Browse the `.yaml` files in this repository
2. Copy or download the component YAML you need
3. Import into your Canvas App using **Paste code** in Power Apps Studio

## Structure

All component YAML files live in the root of this repository for easy browsing and access.
